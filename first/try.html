What is HTML

HTML stands for HyperText Markup Language. It's a markup language that web developers use to stratcure and describe the content of a webpage (not a programming language)

HTML consits of elements that describe different types of content: Paragrph, links, headings, imange, videos, etc. web browser understand HTML and render HTML code as website.

In HTML, each element is made up of 3 parts:

  1. The Opening tag
  2. The Closing tag
  3. The actual elements

You can learn more at MDN Web Docs.
